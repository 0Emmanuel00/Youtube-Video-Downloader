# Youtube Video Downloader

Ce projet permet de télécharger des vidéos YouTube avec une interface simple.

## Prérequis

Avant de lancer le programme, tu dois installer les dépendances suivantes :

1. **Python 3.12**
  Vérifie l'installation avec: python --version
2. Installer yt-dlp
   Pour l'installation de yt-dlp c'est avec: pip install yt-dlp
